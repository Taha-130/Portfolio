Gestion de réservations pour une station de ski

Description

Le projet de base de données "Gestion de réservations pour une station de ski" a été réalisé en binôme dans le cadre du cours SAE 2-04 du premier semestre. L'objectif était de mettre en place une base de données permettant de gérer les réservations pour une station de ski.

Fonctionnalités

  • Enregistrement des utilisateurs (administrateurs, clients, gestionnaires)
  • Gestion des réservations de forfaits, d'hébergements, de cours de ski, etc.
  • Consultation des disponibilités et tarifs
  • Génération de factures et de confirmations de réservation
  • Interface d'administration pour gérer les utilisateurs, les disponibilités, les tarifs, etc.

Technologies utilisées

  • MySQL : Nous avons utilisé MySQL comme système de gestion de base de données relationnel pour stocker et gérer les données de la station de ski.
  • phpMyAdmin : Nous avons utilisé phpMyAdmin, une interface web basée sur PHP, pour administrer et gérer facilement la base de données MySQL.
  • PHP : Nous avons utilisé PHP (Hypertext Preprocessor) comme langage de programmation côté serveur pour développer la logique métier de notre application web.
  • HTML : Nous avons utilisé HTML (Hypertext Markup Language) pour créer la structure et les éléments de contenu de nos pages web.
  • CSS : Nous avons utilisé CSS (Cascading Style Sheets) pour styliser et mettre en forme nos pages web, en appliquant des règles de mise en page, de couleur et de typographie.
  • Python : Nous avons utilisé Python pour développer des scripts pour la manipulation et la transformation des données, notamment pour convertir le script SQL de MySQL en script SQL compatible avec PostgreSQL.

En utilisant cette combinaison de technologies, nous avons pu créer une application web complète et fonctionnelle pour la gestion des réservations d'une station de ski, offrant une interface conviviale et des fonctionnalités avancées.
